### 目录结构
`alpha`：简单版，实现核心逻辑和算法。

`standard`：标准版，美化界面，优化代码。

`beta`：测试版，额外功能补充，一些其他想法。

`plugin`：公共插件目录，目前只有弹窗，后续可能有按钮。

`其他目录`：不用管，用于快速创建项目和打包构建，还有一些脚本。
