## Canvas有趣的小案例
`Canvas` + `ES6`实现经典小游戏和案例。

### 目录结构说明
`alpha`：简单版，实现核心逻辑和算法。

`standard`：标准版，美化界面，优化代码。

`beta`：测试版，额外功能补充，一些其他想法。

### 多说三句

授人以鱼不如授人以渔！我会写博客和录配套视频，逐行手敲代码。
